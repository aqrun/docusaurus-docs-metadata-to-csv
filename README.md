# docusaurus-docs-metadata-to-csv



## Installation

Install the plugin with npm:

```bash
npm install --save @aqrun/docusaurus-docs-metadata-to-csv
```

Add the plugin do `docusaurus.config.js`:

```javascript
module.exports = {
  // ...
  plugins: [
    // ...
    '@aqrun/docusaurus-docs-metadata-to-csv'
  ]
};
```
